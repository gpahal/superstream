# `inspector run`

```txt
Usage: inspector run [options]

Run inspector process

Options:
  -c, --cluster <cluster>                  solana cluster (choices: "mainnet-beta", "devnet", "testnet", "localnet", default: "devnet")
  -j, --json                               show output as JSON
  -d, --debug                              enable debug output
  -w, --wallet <wallet-keypair-json-path>  solana wallet keypair JSON file path
  -h, --help                               display help for command
```
