# `inspector stats`

```txt
Usage: inspector stats [options]

Get stream stats

Options:
  -c, --cluster <cluster>  solana cluster (choices: "mainnet-beta", "devnet",
                           "testnet", "localnet", default: "devnet")
  -j, --json               show output as JSON
  -d, --debug              enable debug output
  -h, --help               display help for command
```
